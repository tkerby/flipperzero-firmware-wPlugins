# Countdown — Numbers Game for Flipper Zero

A game inspired by the "Countdown" numbers round.

Select 6 numbers, choose a 3-digit target,
and the solver will try to reach it using + − × ÷.

## Controls

- D-Pad → Move cursor
- OK → Select number / digit / Solve / Restart
- Back → Go back (or exit in phase 1)

## Build

ufbt build
ufbt launch

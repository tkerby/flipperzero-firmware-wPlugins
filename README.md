# FlipTelegram
Flipper Zero Telegram Client

## Requirements
- WiFi Developer Board, Raspberry Pi, BW16, or ESP32 device flashed with FlipperHTTP v2.1 or higher: https://github.com/jblanked/FlipperHTTP
- 2.4 GHz or 5 GHz WiFi access point
- Telegram Bot Token and Chat ID: Create a bot using BotFather on Telegram and obtain the bot token. Next, add your bot to a chat. Then, get your chat ID by typing in https://api.telegram.org/bot{your_bot_token}/getUpdates in your web browser after sending a message in that chat. If you entered credentials using qFlipper, make sure to add a space after the bot token and chat ID to avoid parsing issues or open the Settings menu in the app and re-save them.

# mitzi-tree-ident
![Logo](images/icon_52x52.png)

**Goal:** Identify a tree by answering questions on your Flipper Zero. 

## Sketch of user flow
In case there is only one question to answer, the user flow would roughly be the following
* **Splash Screen:** Press OK to go to menu
* **Menu Screen:** Use Up/Down to navigate, Right to select
* **Content Screens:** Left for menu
